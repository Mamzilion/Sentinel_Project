# 🛡️ Sentinel - Plateforme d'Examen Sécurisée (IUT-Ngaoundéré)

Sentinel est une solution intégrée de supervision d'examens en réseau local.

## 🚀 Structure du projet
- **/sentinel-server** : API REST Node.js pour la gestion des examens et des logs.
- **/sentinel-agent** : Script Python installé sur les postes étudiants (Lockdown + Webcam).
- **/sentinel-dashboard** : Interface Web de supervision en temps réel pour les enseignants.

## 🛠️ Installation rapide
1. Cloner le projet : `git clone https://github.com/Mamzilion/Sentinel_Project.git`
2. Installer les dépendances serveur : `cd sentinel-server && npm install`
3. Lancer l'agent : `cd sentinel-agent && python main.py`
